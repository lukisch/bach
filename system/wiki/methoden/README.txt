Stand: 2026-02-05

WAS SIND METHODEN?
==================

Methoden sind systematische Vorgehensweisen zur Erreichung
eines bestimmten Ziels. Sie geben Struktur und erhoehen
die Erfolgswahrscheinlichkeit bei komplexen Aufgaben.

  Methode vs. Technik vs. Tool
  ----------------------------
  Methode: Umfassendes Vorgehen mit mehreren Schritten
  Technik: Spezifische Handlungsweise innerhalb einer Methode
  Tool: Hilfsmittel zur Unterstuetzung

  Beispiel:
  - Methode: Design Thinking
  - Technik: Brainstorming (Teil davon)
  - Tool: Post-Its, Timer

METHODENAUSWAHL
===============

  Kriterien fuer die Methodenwahl
  -------------------------------
  1. Ziel: Was soll erreicht werden?
     - Information vermitteln
     - Ideen generieren
     - Entscheidungen treffen
     - Probleme loesen
     - Verhalten aendern

  2. Zielgruppe: Wer nimmt teil?
     - Gruppengroesse
     - Vorkenntnisse
     - Hierarchie
     - Kultur

  3. Rahmenbedingungen: Was ist moeglich?
     - Zeit
     - Raum
     - Material
     - Budget

  4. Phase: Wo im Prozess?
     - Anfang (Aktivierung, Kennenlernen)
     - Mitte (Bearbeitung, Vertiefung)
     - Ende (Ergebnis, Transfer)

  Methoden-Matrix
  ---------------

  | Ziel           | Kleine Gruppe   | Grosse Gruppe    |
  |----------------|-----------------|------------------|
  | Ideen sammeln  | Brainstorming   | Metaplan         |
  | Entscheiden    | Pro/Contra      | Punktabfrage     |
  | Probleme loesen| Fishbone        | World Cafe       |
  | Aktivieren     | Murmelgruppe    | Energizer        |
  | Reflektieren   | Feedback-Runde  | Blitzlicht       |

METHODENKOMBINATION
===================

  Dramaturgie eines Workshops
  ---------------------------

  PHASE 1: ANKOMMEN (10-15%)
  - Check-In (Wie bin ich da?)
  - Erwartungsabfrage
  - Agenda vorstellen

  PHASE 2: AKTIVIEREN (10%)
  - Energizer
  - Warm-Up zur Thematik
  - Erste Assoziationen

  PHASE 3: BEARBEITEN (50-60%)
  - Hauptmethode(n) zum Thema
  - Wechsel zwischen Einzel/Gruppe
  - Visualisierung der Ergebnisse

  PHASE 4: VERDICHTEN (15%)
  - Ergebnisse zusammenfuehren
  - Priorisieren
  - Entscheiden

  PHASE 5: ABSCHLIESSEN (10%)
  - Massnahmenplanung
  - Transfer sichern
  - Check-Out

  Methodenwechsel
  ---------------
  - Alle 20-30 Minuten Format wechseln
  - Abwechslung zwischen:
    - Plenum / Kleingruppe / Einzelarbeit
    - Sprechen / Schreiben / Bewegen
    - Analytisch / Kreativ

WICHTIGE EINZELMETHODEN
=======================

  Brainstorming
  -------------
  Ziel: Viele Ideen in kurzer Zeit
  Regeln:
  1. Quantitaet vor Qualitaet
  2. Keine Kritik waehrend der Sammlung
  3. Wilde Ideen erwuenscht
  4. Auf Ideen anderer aufbauen

  Metaplan-Technik
  ----------------
  Ziel: Strukturierte Ideensammlung in Gruppen
  Material: Karten, Pinwand, Stifte
  Vorgehen:
  1. Frage formulieren
  2. Jeder schreibt Karten (1 Idee pro Karte)
  3. Karten vorstellen und clustern
  4. Cluster benennen
  5. Priorisieren

  World Cafe
  ----------
  Ziel: Wissensaustausch in grossen Gruppen
  Setting: Mehrere Tische, Gastgeber
  Vorgehen:
  1. Fragen auf Tische verteilen
  2. Rotierende Diskussionsrunden
  3. Gastgeber fassen zusammen
  4. Ernte im Plenum

  Design Thinking
  ---------------
  Ziel: Nutzerorientierte Innovation
  Phasen:
  1. Verstehen (Problemraum erkunden)
  2. Beobachten (Nutzer kennenlernen)
  3. Sichtweise definieren (Problem schaerfen)
  4. Ideen finden (Kreativ loesen)
  5. Prototypen (Schnell bauen)
  6. Testen (Feedback holen)

  Open Space
  ----------
  Ziel: Selbstorganisierte Konferenz
  Prinzipien:
  - Wer auch immer kommt, ist richtig
  - Was auch immer geschieht, ist das Einzige
  - Es beginnt, wenn die Zeit reif ist
  - Vorbei ist vorbei
  - Das Gesetz der zwei Fuesse (Gehen erlaubt)

  Fishbone-Diagramm (Ishikawa)
  ----------------------------
  Ziel: Ursachenanalyse
  Kategorien: Mensch, Maschine, Material, Methode,
              Mitwelt, Messung (6M)
  Vorgehen:
  1. Problem als "Kopf" des Fischs
  2. Hauptkategorien als "Graeten"
  3. Ursachen an die Graeten schreiben
  4. Auf Hauptursachen fokussieren

DIGITALE METHODEN
=================

  Online-Moderation
  -----------------
  Besonderheiten:
  - Kuerzere Aufmerksamkeitsspannen
  - Mehr Pausen noetig
  - Technik als Huerde
  - Paralleles Arbeiten schwieriger

  Tools:
  - Miro, Mural (Whiteboards)
  - Mentimeter (Umfragen)
  - Breakout-Rooms (Kleingruppen)
  - Chat fuer Stille

  Hybrid-Formate
  --------------
  Teilnehmer vor Ort + Remote:
  - Technik sorgfaeltig planen
  - Moderatoren fuer beide Welten
  - Remote-Teilnehmer nicht vergessen
  - Gleiche Beteiligungsmoeglichkeiten
