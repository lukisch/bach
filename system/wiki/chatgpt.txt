WAS IST CHATGPT?
----------------
ChatGPT ist der Chatbot von OpenAI, der auf GPT-Modellen basiert.
Aktuell: GPT-5.2 (mit Instant, Thinking, und Pro Varianten)

Verfuegbar als:
  - Web-Chat (chatgpt.com)
  - Mobile Apps (iOS/Android)
  - API (platform.openai.com)
  - ChatGPT for Teams/Enterprise

CONSUMER ABOS
-------------
  Free ($0/Monat)
    - GPT-5.2 Instant (schnellstes Modell)
    - ~10 Nachrichten alle 5 Stunden
    - Danach Fallback auf "Mini" Version
    - Werbung geplant (2026)

  ChatGPT Go ($8/Monat) [NEU 2026]
    - Erweiterter Zugang zu GPT-5.2 Instant
    - Mehr Nachrichten
    - Mehr Uploads
    - Mehr Bildgenerierung
    - Werbung geplant

  ChatGPT Plus ($20/Monat)
    - GPT-5.2 Thinking (tieferes Reasoning)
    - Legacy-Modelle waehlbar
    - Coding-Agent (Codex)
    - Ideal fuer: Schreiben, Research, Datenanalyse
    - WERBEFREI

  ChatGPT Pro ($200/Monat)
    - GPT-5.2 Pro (staerkstes Modell)
    - Maximaler Memory und Context
    - Fruehzugang zu neuen Features
    - Fuer AI Power-User
    - WERBEFREI

API PRICING
-----------
API wird separat von Chat-Abos abgerechnet!
Token-basierte Abrechnung (Pay-as-you-go)

  GPT-5.2 (aktuelle Generation):
    Input:  ~$2-5/Million Tokens (je nach Modell)
    Output: ~$10-20/Million Tokens

  GPT-4o:
    Input:  $2.50/Million Tokens
    Output: $10.00/Million Tokens

  GPT-4o-mini:
    Input:  $0.15/Million Tokens
    Output: $0.60/Million Tokens

  Batch API:
    - 50% Rabatt auf Input UND Output
    - Asynchrone Verarbeitung (bis 24h)

  DALLE-3 (Bildgenerierung):
    - $0.01 (low quality)
    - $0.04 (medium)
    - $0.17 (high, square)

  Whisper (Audio):
    - $0.006/Minute

  Embeddings (text-embedding-3-large):
    - $0.00013/1K Tokens

RESPONSES API

Nicht separat abgerechnet - Tokens werden nach
dem gewaehlten Sprachmodell berechnet.
