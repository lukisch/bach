ZAPIER MCP - KI-AUTOMATISIERUNG UEBER 8.000 APPS
=================================================

Stand: 2026-02-14

WAS IST ZAPIER MCP?
-------------------
Zapier MCP verbindet KI-Modelle (Claude, ChatGPT, Cursor etc.) ueber das Model Context
Protocol mit Zapiers Bibliothek von fast 8.000 Apps und 30.000+ Aktionen. Statt fuer
jede App eine eigene Integration zu bauen, nutzt man Zapiers vorgefertigte Verbindungen.

Status: Open Beta (Stand Feb 2026).

FUNKTIONSWEISE
--------------
1.  KI-Modell sendet Anfrage ueber MCP an Zapier-Server
2.  Zapier uebersetzt in API-Aufrufe an die Ziel-App
3.  Ergebnis wird zurueck an das KI-Modell geliefert

Das KI-Modell kann also z.B. sagen: "Sende eine Slack-Nachricht an #team"
und Zapier fuehrt das aus, ohne dass der Nutzer Slack oeffnen muss.

KOSTEN
------
  - Verfuegbar auf ALLEN Zapier-Plaenen (auch Free)
  - Kostet 2 Zapier-Tasks pro Tool-Aufruf
  - Free Plan: 100 Tasks/Monat = ca. 50 MCP-Aufrufe
  - Starter Plan ($29.99/Mo): 750 Tasks = ca. 375 MCP-Aufrufe
  - Professional Plan ($73.50/Mo): 2.000 Tasks = ca. 1.000 MCP-Aufrufe

EINRICHTUNG CLAUDE DESKTOP
---------------------------
1.  Zapier MCP Dashboard oeffnen: https://zapier.com/mcp
2.  "New MCP Server" waehlen, Client = "Claude"
3.  Gewuenschte Aktionen hinzufuegen (z.B. "Send Slack Message")
4.  App-Verbindung autorisieren (OAuth)
5.  Integration-URL kopieren
6.  In claude_desktop_config.json eintragen:

```json
{
  "mcpServers": {
    "zapier": {
      "command": "npx",
      "args": ["-y", "@zapier/mcp-server"],
      "env": {
        "ZAPIER_MCP_API_KEY": "<dein-api-key>"
      }
    }
  }
}
```

EINRICHTUNG CLAUDE CODE
------------------------
In ~/.claude/.mcp.json eintragen:

```json
{
  "mcpServers": {
    "zapier": {
      "command": "npx",
      "args": ["-y", "@zapier/mcp-server"],
      "env": {
        "ZAPIER_MCP_API_KEY": "<dein-api-key>"
      }
    }
  }
}
```

Alternative: Remote-URL-Endpoint von Zapier Dashboard nutzen.

SICHERHEIT
----------
  - Jede Aktion muss explizit im Zapier Dashboard aktiviert werden
  - On/Off-Toggles fuer jede einzelne Aktion
  - OAuth-basierte App-Autorisierung
  - Aktionen koennen jederzeit deaktiviert werden
  - Enterprise-Plan: Standardmaessig deaktiviert, Admin-Freigabe noetig

LIMITIERUNGEN
-------------
  - Open Beta: Kann sich noch aendern
  - 2 Tasks pro Aufruf (verbraucht Zapier-Kontingent schnell)
  - Nur Aktionen verfuegbar, die Zapier bereits integriert hat
  - ChatGPT: Nur im Developer Mode
  - Enterprise: Braucht Admin-Freigabe
  - Latenz: Abhaengig von Ziel-App-API-Geschwindigkeit
