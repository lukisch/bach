LOBEHUB -- SKILL-MARKETPLACE FUER KI-AGENTEN
=============================================

Stand: 2026-03-12

WAS IST LOBEHUB?
-----------------
LobeHub ist eine Plattform und ein Marketplace fuer KI-Skills und
Agenten-Konfigurationen. Die Plattform crawlt automatisch
oeffentliche GitHub-Repositories und indexiert Skills, die dem
Anthropic/agentskills.io-Standard folgen.

LobeHub fungiert als Vermittler: Entwickler veroeffentlichen Skills
auf GitHub, LobeHub macht sie durchsuchbar und installierbar.


AUTOMATISCHES CRAWLING
-----------------------
LobeHub durchsucht GitHub-Repos nach Skills in diesen Verzeichnissen:

  skills/                  Root-Level Skills-Ordner
  .claude/skills/          Claude-spezifische Skills
  .github/skills/          GitHub-konventioneller Pfad

Jeder Unterordner mit einer SKILL.md wird als eigenstaendiger Skill
erkannt und indexiert. Das Crawling erfolgt automatisch -- kein
manuelles Einreichen noetig. Aenderungen auf GitHub werden zeitnah
im Marketplace reflektiert.


MARKETPLACE-FUNKTIONEN
-----------------------
- Katalog mit 100.000+ indexierten Skills
- Volltextsuche ueber Skill-Namen und Beschreibungen
- SEO-optimierte Skill-Seiten (jeder Skill erhaelt eine eigene URL)
- Automatische Uebersetzung in 16 Sprachen
- Bewertungen und Nutzungsstatistiken
- Kategorisierung nach Domaene und Plattform


INSTALLATION UND AUSLIEFERUNG
-------------------------------
Bei Installation eines Skills ueber LobeHub wird die originale
SKILL.md 1:1 ausgeliefert -- LobeHub veraendert den Inhalt nicht.
Der Nutzer erhaelt exakt das, was der Entwickler auf GitHub
bereitgestellt hat.

Skills sind primaer Instruktions-Sets (Markdown-Dokumente), kein
ausfuehrbarer Code. Die Ausnahme bildet der scripts/-Ordner, der
ausfuehrbare Skripte enthalten kann. Das LLM interpretiert die
Anweisungen aus der SKILL.md und nutzt seine eigenen Tools
(Bash, Read, Edit etc.) zur Umsetzung.


KOMPATIBLE PLATTFORMEN
-----------------------
Skills im Anthropic-Standard sind plattformuebergreifend nutzbar:

  Claude Code          Anthropics CLI-Agent
  Codex CLI            OpenAIs CLI-Agent
  ChatGPT              OpenAIs Chat-Plattform
  Cursor               KI-gestuetzter Code-Editor
  OpenClaw             Open-Source-Alternative

LobeHub ist plattformneutral -- ein dort gelisteter Skill
funktioniert auf jeder kompatiblen Plattform.


BACH-SKILLS AUF LOBEHUB
-------------------------
Folgende BACH-Skills sind bereits auf LobeHub indexiert:

  bach-document          Dokumenten-Erstellung und -Verwaltung
  bach-financial         Finanzanalyse und Buchhaltung
  bach-production        Produktions-Workflows und Deployment

Diese Skills wurden ueber das ellmos-ai GitHub-Profil
veroeffentlicht und automatisch von LobeHub gecrawlt.


LOBEHUB PRICING
----------------
LobeHub bietet gestaffelte Plaene. Credits werden fuer
LLM-Aufrufe ueber die LobeHub-Plattform verbraucht:

  Plan         Preis/Monat    Credits
  ----------   -----------    -----------
  Free         $0             500K Credits
  Basis        $12.90         5M Credits
  Premium      $24.90         15M Credits
  Top-Tier     $49.90         35M Credits
  Enterprise   custom         custom

Alle Bezahlplaene enthalten Zugang zu "Premium Skills".
Premium Skills sind daher kein Differenzierungsmerkmal
zwischen den Plaenen -- der Hauptunterschied liegt im
Credit-Volumen fuer LLM-Aufrufe.

Hinweis: Skills selbst sind kostenlos installierbar und nutzbar.
Die Credits beziehen sich auf die Nutzung der LobeHub-eigenen
LLM-Infrastruktur (gehostete Modelle).


STRATEGIE FUER BACH/ELLMOS
----------------------------
Empfohlener Ansatz fuer die Veroeffentlichung von BACH-Skills:

1. Separates Skill-Repository: ellmos-ai/skills
   - Enthaelt standalone Skills im Anthropic-Format
   - Wird automatisch von LobeHub gecrawlt
   - Unabhaengig vom BACH-Hauptrepo (keine internen Details)

2. Export bestehender BACH-Skills:
   - bach skills export <name> --format anthropic
   - Konvertiert BACH-interne Metadaten in Anthropic-Frontmatter
   - Entfernt BACH-spezifische Abhaengigkeiten

3. Neue Skills direkt im Anthropic-Format:
   - bach skills create <name> --format anthropic
   - Sofort kompatibel mit LobeHub und allen Plattformen

Ziel: BACH-interne Skills bleiben reichhaltiger (DB-Sync, Typ-System,
Abhaengigkeiten). Exportierte Skills sind schlank und portabel.
