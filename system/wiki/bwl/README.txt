DEFINITION
==========
  Die ABWL (Allgemeine Betriebswirtschaftslehre) vermittelt
  grundlegende Kenntnisse ueber die Fuehrung, Steuerung und
  Organisation von Betrieben unabhaengig von ihrer Branche.

  BWL = "Froschperspektive": Wie funktioniert EIN Unternehmen?
  VWL = "Vogelperspektive": Wie funktioniert DIE Wirtschaft?

GRUNDBEGRIFFE
=============
  - Betrieb vs. Unternehmen
  - Gewinn = Ertrag - Aufwand
  - Liquiditaet = Zahlungsfaehigkeit
  - Rentabilitaet = Gewinn / Kapital
  - Produktivitaet = Output / Input
  - Wirtschaftlichkeit = Ertrag / Aufwand
