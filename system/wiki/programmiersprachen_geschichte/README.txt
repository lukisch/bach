GESCHICHTE DER PROGRAMMIERSPRACHEN
==================================

Stand: 2026-01-24
Status: STRUKTUR ANGELEGT

UEBERBLICK
==========
  Evolution der Programmiersprachen von den
  Anfaengen bis zur Gegenwart.

ZEITLICHE EINORDNUNG
====================

  anfaenge/ (1940er-1950er)
  -------------------------
    - Maschinencode, Assembler
    - FORTRAN (1957) - erste Hochsprache
    - COBOL (1959) - Geschaeftsanwendungen
    - LISP (1958) - funktionale Programmierung

  imperative/ (1960er-1970er)
  ---------------------------
    - BASIC (1964) - Einsteigersprache
    - Pascal (1970) - strukturierte Programmierung
    - C (1972) - systemnah, einflussreich

  objektorientierte/ (1980er-1990er)
  ----------------------------------
    - Smalltalk (1980) - reine OOP
    - C++ (1983) - C mit Klassen
    - Java (1995) - plattformunabhaengig
    - Python (1991) - lesbar, vielseitig

  funktionale/ (parallel entwickelt)
  ----------------------------------
    - Haskell (1990) - rein funktional
    - ML, OCaml, F#
    - Erlang (1986) - Concurrency

  skriptsprachen/ (1990er-2000er)
  -------------------------------
    - Perl (1987) - Textverarbeitung
    - PHP (1995) - Web
    - Ruby (1995) - elegant
    - JavaScript (1995) - Browser

  moderne/ (2000er-heute)
  -----------------------
    - C# (2000) - Microsoft/.NET
    - Go (2009) - Google, Concurrency
    - Rust (2010) - Speichersicherheit
    - Kotlin (2011) - modernes Java
    - Swift (2014) - Apple
    - TypeScript (2012) - typisiertes JS

  zukunft/
  --------
    - Trends: Speichersicherheit, Concurrency
    - Low-Code/No-Code
    - KI-gestuetzte Entwicklung
    - Domainspezifische Sprachen

MEILENSTEINE
============
  1957: FORTRAN - erste Hochsprache
  1972: C - Basis fuer Unix und viele Sprachen
  1983: C++ - OOP wird Mainstream
  1995: Java, JavaScript - Web-Aera beginnt
  2010: Rust - Fokus auf Sicherheit
