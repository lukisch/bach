CHIAH - VORGAENGERSYSTEM VON BACH
=================================

WAS IST CHIAH?
--------------
CHIAH (Claude Hybrid Intelligent Assistant Handler) war ein
CLI-first System fuer Claude AI mit Fokus auf strukturierte
Eingaben und Kontext-Management. Es ist einer der drei
Vorgaenger des heutigen BACH-Systems.

CHIAH trug folgende Konzepte zu BACH bei:
- CLI-First Design (bach --command statt GUI)
- Injektoren-System (automatischer Kontext)
- Help-System mit Themen-Dateien
- Context Sources (Wissensquellen)
- Scanner fuer automatische Problem-Erkennung

KERN-KONZEPT: INJEKTOREN
------------------------
CHIAHs wichtigste Innovation waren Injektoren - automatische
Kontext-Hinzufuegung basierend auf Situation:

  strategy_injector    Between-Task Erinnerungen
  context_injector     Memory-System Hinweise
  time_injector        Zeit-Bewusstsein
  between_injector     Aufgaben-Workflow Tipps

Diese laufen heute in BACH via:
  bach --inject list          Aktive Injektoren
  bach --inject toggle NAME   Ein/Ausschalten

UNTERSCHIED ZU BACH
-------------------
CHIAH war CLI-fokussiert. BACH vereint die besten
Konzepte aus drei Systemen:

  BATCH    + CHIAH       + recludOS   = BACH
  (Hub,    + (CLI,       + (Agenten,  = (Best-of
   Sessions) Injektoren)   Distribution)  aller drei)

DIE FORMEL
----------
  BACH = Best-of(_BATCH + _CHIAH + recludOS)
  ATI  = BACH + Delta (Software-Entwickler-Spezialisierung)

LEGACY-STATUS
-------------
CHIAH wird nicht mehr aktiv entwickelt. Alle wichtigen Features
wurden nach BACH migriert:

  [x] CLI-Befehle         -> bach.py mit --command Syntax
  [x] Injektoren          -> tools/injectors.py
  [x] Help-System         -> docs/docs/docs/help/*.txt
  [x] Context Sources     -> In Memory-System integriert
  [x] Scanner             -> tools/tool_scanner.py

HISTORISCHER KONTEXT
--------------------
CHIAH entstand parallel zu BATCH mit anderem Fokus. Waehrend
BATCH Session-Management betonte, fokussierte CHIAH auf
strukturierte CLI-Interaktion und automatische Kontext-Hilfe.
2026 wurden beide in BACH vereint.
