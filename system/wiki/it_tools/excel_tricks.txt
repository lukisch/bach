DIE WICHTIGSTEN SHORTCUTS
=========================

  NAVIGATION
  ----------
    Strg + Pfeiltasten    Ans Ende der Daten springen
    Strg + Pos1           Zum Anfang (A1)
    Strg + Ende           Zur letzten Zelle mit Daten
    Strg + Bild ab/auf    Zwischen Tabellenblaettern

  AUSWAHL
  -------
    Strg + Shift + Pfeil  Bis Datenende markieren
    Strg + Leertaste      Ganze Spalte markieren
    Shift + Leertaste     Ganze Zeile markieren
    Strg + A              Alles markieren

  BEARBEITUNG
  -----------
    Strg + C / V / X      Kopieren / Einfuegen / Ausschn.
    Strg + Z / Y          Rueckgaengig / Wiederholen
    F2                    Zelle bearbeiten
    F4                    Letzte Aktion wiederholen
    Strg + D              Nach unten ausfuellen
    Strg + R              Nach rechts ausfuellen

  FORMATIERUNG
  ------------
    Strg + 1              Zellen formatieren (Dialog)
    Strg + Shift + 1      Zahlenformat
    Strg + Shift + 4      Waehrungsformat
    Strg + Shift + 5      Prozentformat
    Strg + B / I / U      Fett / Kursiv / Unterstrichen

  SPEZIAL
  -------
    Strg + T              In "Intelligente Tabelle" wandeln
    Alt + =               Autosumme einfuegen
    Strg + ;              Aktuelles Datum einfuegen
    Strg + Shift + ;      Aktuelle Uhrzeit einfuegen
    F9                    Alle Formeln neu berechnen

POWER-TOOLS
===========

  PIVOT-TABELLEN
  --------------
    Das maechtigste Analysetool in Excel!
    Drag & Drop fuer Auswertungen ohne Formeln.

    Erstellen:
      1. Daten markieren
      2. Einfuegen → PivotTable
      3. Felder in Bereiche ziehen

    Einsatz:
      - Daten zusammenfassen
      - Gruppieren, Filtern
      - Mehrere Dimensionen analysieren

  BEDINGTE FORMATIERUNG
  ---------------------
    Start → Bedingte Formatierung

    Heatmaps erstellen:
      - Farbskalen (rot-gelb-gruen)
      - Datenbalken
      - Symbolsaetze (Ampeln)

    Regeln:
      - Groesser/kleiner als
      - Text enthaelt
      - Doppelte Werte
      - Eigene Formeln

  BLITZVORSCHAU (FLASH FILL)
  --------------------------
    Strg + E

    Excel erkennt Muster automatisch!

    Beispiel:
      Spalte A: "Max Mustermann"
      Spalte B: Du tippst "Max" und Strg+E
      → Excel fuellt automatisch alle Vornamen

    Funktioniert fuer:
      - Namen aufteilen
      - Formate aendern
      - Texte kombinieren

  INTELLIGENTE TABELLEN
  ---------------------
    Strg + T

    Vorteile:
      - Automatische Erweiterung
      - Strukturierte Verweise (@Spaltenname)
      - Automatische Formatierung
      - Einfache Filter und Sortierung

DATENBEREINIGUNG
================

  DUPLIKATE ENTFERNEN
  -------------------
    Daten → Datentools → Duplikate entfernen

  DATEN TRANSPONIEREN
  -------------------
    Kopieren → Inhalte einfuegen → Transponieren

  TEXT IN SPALTEN
  ---------------
    Daten → Text in Spalten
    Trennzeichen waehlen (Komma, Semikolon, Tab)

  LEERE ZELLEN FUELLEN
  --------------------
    Markieren → F5 → Inhalte → Leerzellen
    = Zelle darueber → Strg + Enter

POWERQUERY (DATEN ABRUFEN)
==========================
  Daten → Daten abrufen und transformieren

  Vorteile:
    - Daten aus vielen Quellen laden
    - Transformationen aufzeichnen
    - Wiederholbare Datenaufbereitung
    - Mehrere Dateien zusammenfuehren

WEITERE HACKS
=============

  SCHNELLANALYSE
  --------------
    Daten markieren → Strg + Q
    Sofort-Auswertungen, Diagramme, Formatierung.

  NAMENSMANAGER
  -------------
    Formeln → Namensmanager
    Bereiche benennen fuer lesbare Formeln.
    =SUMME(Umsatz) statt =SUMME(A1:A100)

  FEHLER FINDEN
  -------------
    Formeln → Formelueberwachung → Spur zum Vorgaenger
    Zeigt, welche Zellen in eine Formel eingehen.

  DATENUEBERPRÜFUNG
  -----------------
    Daten → Datenueberprüfung
    Dropdown-Listen erstellen, Eingaben beschraenken.
