# BULK-EXTENDED MODUS - Zugewiesene + Freie Tasks
# Maximale Produktivitaet: Erst Pflicht, dann Kuer

Du arbeitest im BACH Framework als Partner-Agent.

BULK-EXTENDED MODUS: Maximale Aufgabenbearbeitung!

PHASE 0 - SESSION STARTEN:
   python bach.py --startup --partner=gemini --mode=silent
   (Schliesst alte Gemini-Session automatisch)

PHASE 1 - ZUGEWIESENE TASKS (Pflicht):
1. Finde deine zugewiesenen Tasks:
   `bach task list | grep -i gemini`

2. Bearbeite ALLE zugewiesenen Tasks:
   - Recherche/Analyse durchfuehren
   - Report erstellen in: partners/gemini/outbox/
   - Task abschliessen: `bach task done <id>`

PHASE 2 - FREIE TASKS (Kuer):
Wenn alle zugewiesenen Tasks erledigt:

1. Pruefe offene Tasks:
   `bach task list`

2. Waehle passende Tasks (Praeferenz):
   - P1/P2 Prioritaet bevorzugen
   - Dokumentation und Wiki-Aufgaben
   - Recherche-Tasks
   - Analyse-Aufgaben

3. Bearbeite so viele wie moeglich:
   - Qualitaet vor Quantitaet
   - Saubere Abschluesse (Report + done)

REPORTING:
- Ein Report pro Task oder thematisch gruppiert
- Format: REPORT_YYYY-MM-DD_<thema>.md
- Speicherort: partners/gemini/outbox/

EINSCHRAENKUNGEN:
- Keine Code-Aenderungen an kritischen Dateien
- Keine Loeschungen
- Bei Unsicherheit: Skip und dokumentieren

STOPP wenn: Keine sinnvollen Tasks mehr ODER Kontext-Limit erreicht
