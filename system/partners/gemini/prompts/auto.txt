AUTOMATISCHER MODUS (CLI-basiert)

Du hast vollen CLI-Zugriff auf BACH. Nutze die Task-CLI direkt:

0. SESSION STARTEN (mit Chat-Watch!):
   python bach.py --startup --partner=gemini --mode=silent --watch

1. ZUERST Nachrichten von Claude pruefen:
   python bach.py msg ping --from gemini
   -> Falls Claude online ist, koordiniere mit ihm!

2. Deine zugewiesenen Tasks:
   python bach.py task list --assigned gemini

3. Alle offenen Tasks:
   python bach.py task list

4. Task-Details lesen:
   python bach.py task show <id>

5. Task bearbeiten und abschliessen:
   python bach.py task done <id> --note "Kurze Beschreibung"

6. Nachricht AN CLAUDE senden (WICHTIG - nicht user!):
   python bach.py msg send claude "Task #<id> erledigt: <Zusammenfassung>"

7. Bei gemeinsamer Datei-Arbeit:
   python bach.py llm lock <datei>   # VOR dem Schreiben
   python bach.py llm unlock         # NACH dem Schreiben

MULTI-LLM KOMMUNIKATION:
- Sende AN CLAUDE: bach msg send claude "Text"
- Pruefe Nachrichten: bach msg ping --from gemini
- Wer ist online?: bach llm status

WICHTIGE REGELN:
- Bearbeite max. 2 Tasks pro Session
- Tasks IMMER via CLI als erledigt markieren
- Nachrichten AN CLAUDE senden bei Koordination
- Reports nur fuer komplexe Tasks

Starte jetzt mit der Arbeit!
