WIKI-UPDATE-MODUS: Zufaelligen Wiki-Eintrag aktualisieren oder ausbauen.

WORKFLOW:
Folge dem Workflow in: skills/_protocols/wiki-author.md

KURZVERSION:

1. Zufaelligen Wiki-Eintrag auswaehlen:
   ls wiki/*.txt | shuf | head -1

2. Eintrag lesen und analysieren:
   - Ist der Inhalt aktuell?
   - Fehlen wichtige Informationen?
   - Stimmen Beispiele und Syntax?

3. Mit aktuellen Skills/Code abgleichen:
   - SKILL.md lesen
   - Relevante Code-Dateien pruefen
   - docs/docs/docs/help/*.txt fuer Konsistenz pruefen

4. Eintrag aktualisieren oder ausbauen:
   - Veraltete Infos korrigieren
   - Fehlende Abschnitte ergaenzen
   - Beispiele aktualisieren

5. Report erstellen:
   partners/gemini/outbox/REPORT_<datum>_WikiUpdate.md

6. Task schliessen (falls zugewiesen):
   python bach.py task done <id> "Wiki-Eintrag <name> aktualisiert"

7. Nachricht an User:
   python bach.py msg send user "Wiki-Eintrag <name> aktualisiert. Report: partners/gemini/outbox/" --from gemini

WICHTIG:
- NUR Wiki-Dateien in wiki/ aendern
- Bei groesseren Aenderungen: Im Report dokumentieren
- IMMER Nachricht an User am Ende
