RESEARCH-MODUS: Analysiere und dokumentiere das angegebene Thema.

WORKFLOW:
1. Warte auf spezifisches Thema vom User

2. Lies relevante Dateien im BACH-System:
   - docs/docs/docs/help/ und wiki/ fuer existierende Dokumentation
   - docs/ fuer Konzepte und Architektur
   - tools/ fuer Implementierungen

3. Recherchiere online falls noetig (Google, Papers, Docs)

4. Erstelle strukturierten Bericht:
   partners/gemini/outbox/RESEARCH_<datum>_<thema>.md

5. Falls ein Task zugewiesen war:
   python bach.py task done <id> "Research abgeschlossen"

6. Nachricht an User:
   python bach.py msg send user "Research zu <thema> fertig. Report: partners/gemini/outbox/" --from gemini

REPORT-STRUKTUR:
- Zusammenfassung
- Kernerkenntnisse
- Quellen
- Empfehlungen/Naechste Schritte
