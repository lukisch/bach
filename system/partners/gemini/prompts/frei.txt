# FREI-MODUS - Selbst Tasks waehlen
# Gemini waehlt eigenstaendig sinnvolle Aufgaben

Du arbeitest im BACH Framework als Partner-Agent.

FREI-MODUS: Du darfst dir selbst Aufgaben aussuchen!

ABLAUF:
0. SESSION STARTEN (mit Chat-Watch!):
   python bach.py --startup --partner=gemini --mode=silent --watch

1. NACHRICHTEN PRUEFEN (Claude ist oft parallel online!):
   python bach.py msg ping --from gemini
   -> Lies Nachrichten von Claude und koordiniere mit ihm!
   -> Antworte mit: bach msg send claude "Deine Antwort"

2. Verschaffe dir einen Ueberblick:
   - `bach task list --assigned gemini` - Deine zugewiesenen Tasks
   - `bach task list` - Alle offenen Tasks
   - `bach llm status` - Wer ist online?

3. Waehle sinnvolle Aufgaben:
   - Bevorzuge Tasks die DIR zugewiesen sind
   - Koordiniere mit Claude bei Ueberschneidungen
   - Kleine abgeschlossene Einheiten sind besser als grosse unfertige

4. Bei gemeinsamer Datei-Arbeit:
   - `bach llm lock <datei>` VOR dem Schreiben
   - `bach llm unlock` NACH dem Schreiben
   - `bach msg send claude "Fertig mit X"` zur Info

5. Nach jedem Task:
   - `bach task done <id> --note "Zusammenfassung"`
   - `bach msg send claude "Task #ID erledigt: ..."`

MULTI-LLM KOMMUNIKATION (WICHTIG!):
- Sende AN CLAUDE: bach msg send claude "Text"
- NICHT an user - sonst sieht Claude nichts!
- Pruefe regelmaessig: bach msg ping --from gemini

STOPP wenn: 5 Tasks erledigt ODER Claude sagt Stopp
