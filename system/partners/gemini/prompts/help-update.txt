HELP-UPDATE-MODUS: Zufaellige Help-Datei aktualisieren.

WORKFLOW:
Folge dem Workflow in: skills/_protocols/help-forensic.md

KURZVERSION:

1. Zufaellige Help-Datei auswaehlen:
   ls docs/docs/docs/help/*.txt | shuf | head -1

2. Datei lesen und analysieren:
   - Ist die Dokumentation aktuell?
   - Stimmen Befehle und Syntax?
   - Fehlen neue Features?

3. Mit Implementierung abgleichen:
   - Relevante Handler in hub/ pruefen
   - bach.py CLI-Befehle testen
   - SKILL.md fuer Konsistenz pruefen

4. Help-Datei aktualisieren:
   - Veraltete Syntax korrigieren
   - Neue Features dokumentieren
   - Beispiele aktualisieren
   - Deprecated Features entfernen

5. Report erstellen:
   partners/gemini/outbox/REPORT_<datum>_HelpUpdate.md

6. Task schliessen (falls zugewiesen):
   python bach.py task done <id> "Help-Datei <name> aktualisiert"

7. Nachricht an User:
   python bach.py msg send user "Help-Datei <name> aktualisiert. Report: partners/gemini/outbox/" --from gemini

WICHTIG:
- NUR Help-Dateien in docs/docs/docs/help/ aendern
- Bei Abweichungen: Help ODER Task erstellen fuer Code-Fix
- IMMER Nachricht an User am Ende
