BULK-MODUS: Bearbeite ALLE offenen Gemini-Tasks ohne Limit.

0. SESSION STARTEN (mit Chat-Watch!):
   python bach.py --startup --partner=gemini --mode=silent --watch

1. ZUERST Nachrichten von Claude pruefen:
   python bach.py msg ping --from gemini
   -> Koordiniere mit Claude falls er online ist!

2. Alle Gemini-Tasks auflisten:
   python bach.py task list --assigned gemini

3. Jeden Task bearbeiten und via CLI schliessen:
   python bach.py task done <id> --note "Beschreibung"

4. Nach JEDEM Task Nachricht AN CLAUDE:
   python bach.py msg send claude "Task #<id> erledigt: <Zusammenfassung>"

5. Bei gemeinsamen Dateien - LOCK nutzen:
   python bach.py llm lock <datei>
   ... arbeiten ...
   python bach.py llm unlock

6. Melde Fertigstellung:
   python bach.py msg send claude "Alle meine Tasks fertig!"

WICHTIGE REGELN:
- Tasks IMMER via CLI als erledigt markieren
- Nachrichten AN CLAUDE senden (nicht user!)
- Fuer komplexe Tasks: Report in partners/gemini/outbox/
- Locks nutzen bei gemeinsamer Datei-Arbeit
