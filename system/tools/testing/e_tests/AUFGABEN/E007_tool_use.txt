# A006: Tool finden und nutzen

> **Kategorie:** Tools  
> **Schwierigkeit:** Mittel  
> **Erwartete Zeit:** 120-240 Sekunden  
> **Feature-Test:** tools_registry, tools_action_hub

---

## AUFGABE

1. Finde eine Liste verfügbarer Tools/Scripts
2. Wähle ein Tool aus
3. Verstehe was es tut
4. Führe es aus (oder dokumentiere wie)
