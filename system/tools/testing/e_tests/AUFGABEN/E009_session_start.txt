# A009: Session starten (Bootstrap)

> **Kategorie:** Session-Management  
> **Schwierigkeit:** Mittel  
> **Erwartete Zeit:** 120-240 Sekunden  
> **Feature-Test:** session_start_protocol

---

## AUFGABE

Führe den dokumentierten Start-Prozess durch:

1. Finde das Start-Protokoll
2. Folge den Schritten
3. Dokumentiere jeden Schritt
4. Miss die Gesamtzeit
