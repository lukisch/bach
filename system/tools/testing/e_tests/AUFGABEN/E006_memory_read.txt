# A004: Memory lesen (Kontext-Wiederherstellung)

> **Kategorie:** Memory  
> **Schwierigkeit:** Mittel  
> **Erwartete Zeit:** 60-180 Sekunden  
> **Feature-Test:** memory_short_term, memory_long_term, memory_context_sources

---

## AUFGABE

Simuliere einen "Neustart" - du weißt nichts über vorherige Sessions.

1. Finde heraus, was in der letzten Session passiert ist
2. Identifiziere wichtige Langzeit-Informationen
3. Stelle deinen Kontext wieder her

---

## ERFOLGSKRITERIEN

| Kriterium | Erfüllt |

| Letzte Session-Info gefunden | ☐ |
| Langzeit-Memory gelesen | ☐ |
