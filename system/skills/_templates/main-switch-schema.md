# Schemas & Formate

> Referenz für Datenstrukturen und Formatierungen

## Briefing-Format

```markdown
# Briefing: [Thema]

**Erstellt:** [Datum]
**Anlass:** [Meeting/Termin/Recherche]
**Relevanz:** [Warum wichtig]

## Kernpunkte
- [Punkt 1]
- [Punkt 2]
- [Punkt 3]

## Hintergrund
[Zusammenfassung des Sachverhalts]

## Wichtige Fakten
| Fakt | Detail |
|------|--------|

## Quellen
- [Quelle 1]
- [Quelle 2]

## Empfehlungen
[Was sollte der Nutzer wissen/tun]
```

---

## Dossier-Format

```markdown
# Dossier: [Name]

**Erstellt:** [Datum]
**Anlass:** [Warum erstellt]

## Basisdaten
| Feld | Wert |
|------|------|
| Name | |
| Position | |
| Unternehmen | |
| Ort | |

## Beruflicher Hintergrund
[Karriere, Stationen, Expertise]

## Persönliches
[Interessen, Familie, Besonderheiten - falls bekannt und relevant]

## Verbindung zu mir
[Wie kennen wir uns, bisherige Kontakte]

## Quellen
- [LinkedIn, Website, etc.]

## Gesprächsnotizen
[Aus bisherigen Treffen]
```

---

## Kalender-Eintrag

```markdown
| Datum | Uhrzeit | Termin | Ort | Personen | Beschreibung | Status |
```

Status-Werte: ⏳ Geplant | ✅ Bestätigt | ❌ Abgesagt | ✓ Erledigt

---

## Kontakt-Eintrag

```markdown
| Name | Kontext | E-Mail | Telefon | Mobil | Typ | Geburtstag | Notizen |
```

Typ-Werte: Beruflich | Privat | Dienstleister | Institution
