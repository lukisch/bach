CONTEXT - Kontext-Suche und Longterm-Memory
=============================================

BESCHREIBUNG
------------
Der Context Handler ermöglicht umfassende Suche und Verwaltung von Kontext-
informationen. Er durchsucht Memory-Archive, Chat-Historien, Task-Beschreibungen
und System-Dateien. Ideal für die automatische Kontextladung vor neuen Sessions.

OPERATIONEN
-----------
  bach context auto                    Automatisch relevanten Kontext laden
                                       (letzte Memory-Einträge, Prioritäts-Tasks,
                                        Lessons Learned, Warnungen)

  bach context search BEGRIFF          Durchsuche alle Quellen mit Suchbegriff
  [--from DATUM] [--to DATUM]          Unterstützt AND/OR/NOT-Logik
  [--scope BEREICH]                    Scope: 'all'|'longterm'|'system'

  bach context longterm                Zeige Memory-Archiv sortiert nach Zeit
  [--from DATUM] [--to DATUM]          Datum-Filter im Format YYYY-MM-DD
                                       Zeigt max. 20 Einträge + Zähler

  bach context recent [N]              Zeige letzte N Memory-Einträge
                                       Default: 5 wenn N nicht angegeben

BEISPIELE
---------
  # Auto-Kontext laden (Sessionstart)
  bach context auto

  # Nach Begriffen suchen
  bach context search "deployment pipeline"
  bach context search "FEHLER regression" --from 2026-02-01 --to 2026-03-01
  bach context search "Python unittest" --scope system

  # Memory durchsuchen
  bach context longterm
  bach context longterm --from 2026-02-15

  # Letzte Einträge
  bach context recent
  bach context recent 10

DATEIEN
-------
  Handler: hub/context.py
  Memory-Archiv: memory/*.md
  Task-Verwaltung: memory/tasks.json
  Lessons: docs/help/lessons.txt
  Log-Verzeichnis: data/logs/

SIEHE AUCH
----------
  bach search     - Dateien im Workspace durchsuchen
  bach memory     - Memory-Einträge erstellen/verwalten
  bach status     - Aktuelle Session-Informationen
