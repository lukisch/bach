HANDLER-NAME
============
inject -- Injektor-Steuerung und Aufgaben-Zuweisung

BESCHREIBUNG
============
Der inject-Handler verwaltet das Injektor-System fuer automatisierte
Kontext-Einspeisungen und Task-Zuweisung. Injektoren sind Plugins, die
bei bestimmten Triggern (Woerter, Zeitpunkte, Task-Abschluss) automatisch
hilfreiche Hinweise oder Aktionen einspeisen.

OPERATIONEN
===========
status
  Zeigt den Status aller Injektoren (aktiv/inaktiv).
  Syntax: --inject status

toggle <name>
  Schaltet einen Injektor an oder aus.
  Syntax: --inject toggle <name>

  Verfuegbare Injektoren:
    strategy   Hilfreiche Gedanken bei Trigger-Woertern
    context    Kontext-Hinweise bei Stichworten
    time       Regelmaessige Zeit-Updates (Timebeat)
    between    Auto-Erinnerung nach Task-Done

task <minutes>
  Weist eine neue Aufgabe mit angegebener Dauer zu.
  Syntax: --inject task <minutes>
  Standard: 5 Minuten

decompose <id>
  Zerlegt eine existierende Aufgabe in Unterziele.
  Syntax: --inject decompose <task-id>

BEISPIELE
=========
# Status aller Injektoren pruefen
--inject status

# Strategy-Injektor aktivieren
--inject toggle strategy

# 15-Minuten-Task zuweisen
--inject task 15

# Aufgabe mit ID 'abc123' zerlegen
--inject decompose abc123
