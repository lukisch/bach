HANDLER-NAME
============
N8nManagerHandler


BESCHREIBUNG
============
n8n Workflow Manager Integration fuer BACH. Verwaltet n8n-Instanzen und
Workflows ueber REST-API und optionalen MCP-Server. Standard-Port: 5678.

Profile: n8n-manager
Package:  n8n-manager-mcp (optional, via "bach setup n8n" installierbar)
Verbindung: http://localhost:5678 (konfigurierbar)


OPERATIONEN
===========
status      Prueft n8n-Instanz und MCP-Server-Installation
            Gibt Verbindungsstatus an.

list        Listet verfuegbare Workflows auf (via n8n REST API)
            Zeigt ID, Name, Active-Status.

sync        Vorbereitung fuer Workflow-Synchronisation
            Benoetigt MCP-Server. Delegiert an Claude-Code-Tools.


BEISPIELE
=========
handler = N8nManagerHandler()

# Status pruefen
result = handler.handle('status')

# Workflows auflisten
result = handler.handle('list')

# Sync aktivieren
result = handler.handle('sync')

# Handler mit Custom-URL
handler = N8nManagerHandler('http://n8n.example.com:5678')
