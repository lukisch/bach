HANDLER-NAME
============
path - Zeigt und verwaltet BACH-Systempfade


BESCHREIBUNG
============
Der path-Handler ermittelt und zeigt alle wichtigen BACH-Verzeichnisse an. Er
nutzt bach_paths.py als zentrale Quelle (Single Source of Truth). Pfade koennen
lokal ueberschrieben und validiert werden.

Versionsstand: v2.0


OPERATIONEN
===========
show, all           Alle wichtigen Pfade anzeigen (Standard)
list                Alle verfuegbaren Pfadnamen auflisten (gruppiert)
<name>              Spezifischen Pfad anzeigen (mit Zusatzinfo)
set <name> <pfad>   Pfad-Override in Datenbank setzen
overrides           DB-Overrides anzeigen
validate            Alle kritischen Pfade validieren


BEISPIELE
=========
bach path
  -> Alle Pfade (Standard)

bach path root
  -> Hauptverzeichnis

bach path list
  -> Alle verfuegbaren Pfadnamen (gruppiert)

bach path templates
  -> Pfad zum Templates-Verzeichnis mit Details

bach path set wissensdatenbank /mnt/my/path
  -> Override in DB speichern

bach path validate
  -> Validierung durchfuehren

PFAD-HEILUNG / PATH HEALING
===========================
Pfade werden automatisch ueber bach_paths.py verwaltet.
Manuelle Pfad-Reparaturen sind NICHT noetig!

  bach_paths.py berechnet alle Pfade dynamisch relativ zum
  eigenen Standort. Wenn BACH verschoben wird, passen sich
  alle Pfade automatisch an.

Nur Dokumentations-Pfade (in .txt/.md Dateien) muessen bei
Umstrukturierungen aktualisiert werden:

  bach --maintain docs-paths          Dokumentations-Pfade pruefen
  bach --maintain docs-paths --apply  Dokumentations-Pfade korrigieren

Siehe: bach --help tools/path_healer  (Details)
