OLLAMA WORKER - Async Job-Verarbeitung
======================================
Laueft im Hintergrund, verarbeitet Jobs aus pending/ via Ollama,
schreibt Ergebnisse nach completed/.

Usage:

  python ollama_worker.py --daemon # Dauerhaft laufen
